This directory contains (besides this README.txt) four folders (Fibril, Monomer, Collagen, Fibronectin) that contain ITP files (separated by chain) of the simulations discussed in the article.

Fibril;
This folder contains five sub-directories containing SAA3 fibril ITP files (separated by chain) describing the system:

1) control:
Topology file describing the system for each chain.
topol_Protein_chain_A.itp - topol_Protein_chain_F.itp ; topology .itp files for individual SAA3 fibril chains
posre_Protein_chain_A.itp - posre_Protein_chain_F.itp ; position restraint .itp files for individual SAA3 fibril chains

2) R5S:
Topology file describing the system for each chain.
topol_Protein_chain_A.itp - topol_Protein_chain_F.itp ; topology .itp files for individual SAA3 fibril chains
posre_Protein_chain_A.itp - posre_Protein_chain_F.itp ; position restraint .itp files for individual SAA3 fibril chains

topol_Protein_chain_G.itp - topol_Protein_chain_L.itp ; topology .itp files for individual R5S peptides
