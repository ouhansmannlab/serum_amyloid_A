# serum_amyloid_A

This directory contains Supporting Information files and ITP files of all simulations discussed in the article.

Supporting Information - This directory contains (besides the README.txt) four folders (Fibril, Monomer, Collagen, and Fibronectin) that contain files with the atomic coordinates (in PDB format) of SAA structures discussed in the article. An additional folder (Parameter Files) contains mdp files of parameter settings for GROMACS simulations.

ITP Files - This directory contains (besides the README.txt) four folders (Fibril, Monomer, Collagen, and Fibronectin) that contain ITP files (separated by chain) of all simulations discussed in the article.
