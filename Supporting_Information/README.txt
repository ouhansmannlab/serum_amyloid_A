This directory contains (besides this README.txt) four folders (Fibril, Monomer, Collagen, Fibronectin) that contain files with the atomic coordinates (in PDB format) of SAA3 structures discussed in the article and topology files for the simulations. An additional folder (Parameter Files) contains mdp files of parameter settings for GROMACS simulations.

Fibril;
This folder contains five sub-directories containing all SAA3 fibril configurations and topology files:

1) control:
Atomic coordinates (in PDB format) of the start and final configurations of the control SAA3 fibril simulations:
saa3_f_control_1_initial : Initial structure of SAA3 fibril replica 1
saa3_f_control_2_initial : Initial structure of SAA3 fibril replica 2
saa3_f_control_3_initial : Initial structure of SAA3 fibril replica 3
saa3_f_control_1_final   : Final structure of SAA3 fibril replica 1
saa3_f_control_2_final   : Final structure of SAA3 fibril replica 2
saa3_f_control_3_final   : Final structure of SAA3 fibril replica 3
topol.top                : Topology file for SAA3 fibril control simulations
